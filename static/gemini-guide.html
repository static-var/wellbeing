Gemini API key setup

You only need one thing here: a Gemini API key. This page walks you through creating a project in Google AI Studio, making a key, and pasting it into Wellbeing. We handle the model choice on our side so setup stays simple.

Privacy note

When you add your own Gemini key, Wellbeing sends messages to Google's Gemini API with that key. Google processes those requests.

That means Google's own API terms, privacy policy, and data-handling rules apply. We do not control Google's retention or policy changes.

Step-by-step

Open Google AI Studio

Go to aistudio.google.com and sign in with the Google account you want to use.

Create or select a project

Use the project switcher in AI Studio. If you do not already have a project, create one. Your Gemini key and quota will belong to that project.

Open the API key page

Go to the AI Studio API key page for that project. If Google asks you to confirm project access or set anything up first, do that and come back here.

Create a Gemini API key

Click the button to create a key, then copy it right away. Gemini API keys usually start with AIza.

Paste the key into Wellbeing

Go back to onboarding or settings, paste the key, and save. If server-side key storage is enabled, Wellbeing stores it encrypted.

What you provide vs what Wellbeing decides

You provide the API key.

Wellbeing chooses the Gemini model. That keeps setup simpler and avoids making every user pick from a long model list.
